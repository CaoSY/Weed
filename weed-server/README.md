## Getting started

- Make sure you have [Node.js](https://nodejs.org/en/) installed
- Make sure you have [MongoDB](https://www.mongodb.com/) installed
- Run: `npm install`
- Run: `mongod --dbpath=data`
- Run: `node app.js`
